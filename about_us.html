About The Web Extension

What is Our Web Extension?

There is a need for Tagalog context-based profanity detection and censorship programs for Tagalog text content. The proponents attempted to create a web extension using a fine-tuned Tagalog BERT model trained via the manually scraped and annotated Tagalog Twitter Profanity Dataset (TTP). The proposed web extension will give Twitter users a convenient tool to censor contextually-abusive Tagalog profane words within displayed Tagalog text content

Features

Our web extension offers a wide range of features, including:

  • Context-based detection of profanities within statements used in either an Abusive or Non-abusive manner;
  • Censorship of profanities that are Abusive in context;
  • Feedback/Report feature to keep track of misclassifications and system performance

Purpose of the Web Extension

Our web extension is designed to detect and censor Tagalog profanities on Twitter. It is intended to promote a respectful online environment and to prevent abusive and harmful language from being used in online communication. The web extension is limited to fourteen Tagalog profane words - specifically pakyu, pucha, bobo, kupal, pakshet, putangina, tanga, ulol, punyeta, bwiset, gago, puta, tarantado, and tangina. Any words that are not included will not be detected nor censored by the web extension.
