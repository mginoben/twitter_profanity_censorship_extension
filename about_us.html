What is CussControl?

CussControl provides Twitter users a convenient tool that censors posted Tagalog tweets that contains contextually-abusive profane/swear words.There is a need for Tagalog context-based profanity detection and censorship programs for Tagalog text content. The web extension was developed using a fine-tuned Tagalog BERT model trained on manually scraped and annotated Tagalog Twitter Profanity Dataset (TTP).

Features

  • Context-based detection of profanities within statements used in either an Abusive or Non-abusive manner;
  • Censorship of profanities that are Abusive in context;
  • Feedback/Report feature to keep track of misclassifications and system performance
